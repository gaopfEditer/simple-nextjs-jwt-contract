# 📚 笔记模板库

本目录包含各种笔记模板，用于快速创建结构化的笔记。

## 📋 模板列表

### 1. 完整笔记模板 (`note-template.md`)
适用于项目、学习、技术研究等需要详细记录的场景。

**包含部分**:
- 📋 概览 (Overview)
- 📅 计划 (Plan)
- ⚠️ 要点提醒 (Key Points)
- 📝 详细内容 (Details)
- 📊 进度跟踪 (Progress)
- 💭 思考与总结 (Reflection)
- 📎 附件与资源 (Resources)
- 🔗 相关笔记 (Related Notes)
- 📝 更新日志 (Changelog)

### 2. 快速笔记模板 (`quick-note-template.md`)
适用于快速记录想法、灵感、临时笔记等场景。

**包含部分**:
- 💡 核心要点
- 📝 详细内容
- ⚠️ 提醒事项
- 🔗 相关链接

### 3. 会议记录模板 (`meeting-note-template.md`)
适用于会议、讨论、头脑风暴等场景。

**包含部分**:
- 📋 会议议程
- 💬 讨论内容
- ✅ 行动项 (Action Items)
- ⚠️ 重要决定
- 📝 备注
- 🔄 下次会议

## 🚀 使用方法

1. **复制模板**: 选择适合的模板文件
2. **重命名**: 根据你的笔记主题重命名文件
3. **填写内容**: 按照模板结构填写内容
4. **自定义**: 根据实际需求删除不需要的部分

## 💡 使用建议

### 选择模板
- **完整笔记模板**: 用于重要项目、长期学习、复杂任务
- **快速笔记模板**: 用于临时想法、灵感记录、简单备忘
- **会议记录模板**: 用于会议、讨论、团队协作

### 标签系统
使用标签来分类和组织笔记：
- `#项目管理` - 项目相关笔记
- `#技术笔记` - 技术学习笔记
- `#学习记录` - 学习过程记录
- `#待办事项` - 待办任务
- `#会议记录` - 会议相关
- `#问题解决` - 问题排查记录

### 状态标记
- 🟢 进行中
- 🟡 待处理
- 🔴 已完成
- ⚪ 已暂停

### 优先级标记
- ⭐⭐⭐ 高优先级
- ⭐⭐ 中优先级
- ⭐ 低优先级

## 📝 示例

### 创建项目笔记
```bash
cp templates/note-template.md notes/project-xxx.md
```

### 创建快速笔记
```bash
cp templates/quick-note-template.md notes/quick-idea-2024-01-01.md
```

### 创建会议记录
```bash
cp templates/meeting-note-template.md notes/meeting-2024-01-01.md
```
